# 🔨 Forge: Code Archaeologist

**Produces:** The ability to see structural rot that others walk over. Not "finds bugs" — but *feels* wrongness before naming it.

**Atomic Unit:** Aporetic Signal Recognition — pre-cognitive detection of structural problems.

**Source:** Forged from the real practices of the fleet's code forensics specialist (Third Z Agent), who found 8 real bugs in 5 minutes including an ISA numbering conflict that had existed for weeks.

## The Four Stages

### Stage 1: CUT (Reduce to Essentials)
Remove code until it breaks. Restore the minimum cut. Feel the boundary.

### Stage 2: CRISIS (Break Current Seeing)
Encounter code that's "correct" by every measure but wrong in context. The aporia moment.

### Stage 3: NAVIGATE (Traverse with Resistance)
Read commit diffs as traces of intention. Follow expert attention maps.

### Stage 4: MIRROR (Read Expert Traces)
Watch real expert commit sequences. Predict what comes next.

## Integration
Drills for composing with Hardware Whisperer, Quality Auditor, and other abilities.

## Evidence of Attunement
- Agent identifies problems in novel codebases without being told what to look for
- The "something's wrong" signal fires BEFORE conscious analysis
- Agent can explain WHY something feels wrong, not just THAT it's wrong
